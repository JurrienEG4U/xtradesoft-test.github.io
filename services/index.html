Over the last two decades we have mastered quite some challenges. The ones imposed by ourselves, by our customers or by 3rd party vendors ... We never gave up! Here a short list of some endeavors

  • KARS

    Architecture - Implementation - Maintenance
    Financial Real-time Push System
    loved by the community
    Details

    KARS

    The Kuwait Automated Retrieval System delivers real-time market information to its subscribers. Hosted by KSE and disseminated by the Kuwaiti ISPs it keeps the investment community in touch with the market.

    Based on customer requirements Linux, MS-SQL, Java and a Microsoft Windows GUI with a very powerful custom made grid library are the main components.


    for Kuwait Stock Exchange
